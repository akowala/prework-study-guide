Hypertext Markup Language (HTML)

  • HTML is the "frame" of developing a webpage.
  • Declarations are not case-sensitive; HTML elements are case sensitive.
  • The head element is a container for metadata. Meta elements contain information about the page that is used by the browser. The user won't see information in the head or meta elements.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.

    • Metadeta Tags

  • Title defines the title for a webpage. This is extremely important for Search Engine Optimization.
  • Style defines the CSS styles associated with a webpage.
  • Base defines the base URL for a webpage.
  • Link connects an external resource to the HTML document.
  • Meta defines metadata such as the character set, description, keywords, author, and viewport.
  • Script embeds client-side scripts in the HTML document.

    • Cascading Style Sheets (CSS)

    • In CSS, you describe the presentation of a document written in a markup language such as HTML.
    • In short, CSS handles the look and feel of a web page.
    • A margin indicates how much space we want around the outside of an element.
    • A padding indicates how much space we want around the content inside an element.

    Git

    • GitHub is a code hosting platform for version control and collaboration. It lets you and others work together on projects from anywhere.
    • Remember to "git add -A" in the terminal to stage and commit work to Github.
    • Once changes have been staged, commit to the repository by using "git commit -m" followed by a short and descriptive message about the changes implemented.
    • Use "git status" to keep track of the correct branch.
    • Remember to sync local branches with the remote branch in GitHub. Do this by using "git pull origin main"

    JavaScript

    • A variable is a named container that allows us to store data in our code.
    • Control flow is the order in which a computer executes code in a script.
    • To declare a variable, always use the var keyword. After, give the variable a name.
    • The console.log() method outputs a message to the web console.
    • The console.log method makes it easier for developers to test their code. It helps with debugging codes that are written.
    • Control flow is the order in which a computer executes code in a script.
    • Conditional statements and lops control the order in which code is executed, if at all.
    • A JavaScript Boolean represents one of two values: true or false. If you need to use "yes" or "no" about something, use Boolean.
    • JavaScript statements are composed of: Values, Operators, Expressions, Keywords, and Comments.
